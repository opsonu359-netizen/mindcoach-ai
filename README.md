# MindCoach AI
An AI-powered stress management web app.

## Live Demo
https://mindcoach-app.tiiny.site

## Features
- AI chat for stress management
- Mood tracking
- Urge surfing timer
- Voice output
- Breathing exercises

## Tech Stack
- HTML, CSS, JavaScript
- Cloudflare Workers (Serverless)
- OpenRouter AI API
- Hosted on tiiny.host
